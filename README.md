### 1. Adaptive Interface
- **Branch Switcher:** Instantly toggle between Science (Cyan), Arts (Amber), and Commerce (Emerald) themes.
- **Glassmorphism:** High-end dark UI with `backdrop-blur-xl` and border lighting.
- **Responsive:** Fully optimized for mobile and desktop.

### 2. Student Toolkit
- **CBCS CGPA Calculator:** Calculate Semester GPA based on official credit weights.
- **Attendance Tracker:** Visual dashboard to track the 75% attendance criteria.
- **Math Suite:** Live LaTeX rendering and Desmos graphing integration.
- **Smart Library:** Integrated PDF viewer with zoom controls.
- **Bilingual Cheat Sheets:** Split-view summaries (Hindi/English) for exam prep.
